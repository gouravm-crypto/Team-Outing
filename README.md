# 🌿 Team Outing 2026 — Destination Planner

A beautiful, interactive travel planning website to help your team choose between **Thekkady**, **Coorg**, and **Chikmagalur** for your March 28–29 team outing.

## 🚀 How to deploy on GitHub Pages (5 minutes)

1. **Create a new GitHub repository**
   - Go to [github.com/new](https://github.com/new)
   - Name it something like `team-outing-2026`
   - Make it **Public**
   - Click **Create repository**

2. **Upload the file**
   - Click **"uploading an existing file"** on the repo page
   - Drag and drop `index.html`
   - Click **Commit changes**

3. **Enable GitHub Pages**
   - Go to **Settings** → **Pages**
   - Under **Source**, select `Deploy from a branch`
   - Choose `main` branch, `/ (root)` folder
   - Click **Save**

4. **Your site is live!**
   - URL will be: `https://YOUR-USERNAME.github.io/team-outing-2026`
   - Share this link with your team to vote!
